# Descripcion
Sistema de gestión para minimercados desarrollado en Python para la materia de Análisis de Sistemas.

# Objetivos
El objetivo del sistema es automatizar los procesos de ventas, inventario, pagos y generación de reportes dentro de un minimercado.

## Tecnologías utilizadas
- Python
- Tkinter
- SQLite
- bcrypt
- rich

## Módulos del sistema
- Gestión de ventas
- Gestión de inventario
- Gestión de usuarios
- Gestión de pagos
- Reportes
